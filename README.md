## Run Settings
### Package install
If 'ModuleNotFoundError' occurs during project execution, download the necessary package using the pip command.

- Requests
```
> pip install requests
```

- Beautifulsoup4
```
> pip install beautifulsoup4
```

- Flask
```
> pip install flask
```
